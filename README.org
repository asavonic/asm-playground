
* Asm playground

In this repository I practice learn-by-doing approach to get
experience in x86 assembly on Linux.

* Tasks

** Hello world
Print "Hello, world!" to stdout.

** Hello printf
Call printf function from C standard library.

** Loops
Write a loop to compute a value:
#+begin_src c
  int val = 0;
  for (int i = 0; i < 5; ++i) {
    val += 2;
  }
#+end_src
